# Initial Scaffold Archive

This folder preserves the early smoke-test scaffold that was useful at the beginning of the project. It is not the current production pipeline.

The active project workflow now lives in the top-level `scripts` package, with supporting documentation in `docs` and committed review outputs in `output`.

Archived contents:

- `hmda_plan/`: early package used for manifest-only smoke-test planning.
- `configs/smoke_test.example.json`: early example configuration.
- `docs/smoke-test-design.md`: early design note.
- `tests/test_smoke_plan.py`: tests for the archived scaffold.
